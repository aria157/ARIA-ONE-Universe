# KEYX Whitepaper Files

This directory contains the official KEYX whitepaper in multiple formats:

## Files

- **KEYX_Whitepaper_Web.md** - Web-ready markdown version with full details and links
- **KEYX_Whitepaper_Cover.svg** - Cover art with ARIA ONE geometry (circle-triangle-A)
- **generate-whitepaper.py** - Python script to generate PDF version using ReportLab

## Generate PDF

To create the PDF version:

```powershell
# Install ReportLab
pip install reportlab

# Run the generator
python generate-whitepaper.py
```

This will create `KEYX_Whitepaper_v1.pdf` in this directory.

## Usage

- **Web:** Use the markdown file for GitHub, websites, or documentation
- **PDF:** Share professionally formatted version with investors/partners
- **SVG:** Use as cover image for social media, documents, or branding
